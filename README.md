## Git Workflow

1. Initialized a Git repository using `git init`
2. Added project files using `git add .`
3. Created commits using `git commit`
4. Created a feature branch using `git checkout -b feature-update`
5. Pushed the branch to GitHub
6. Created a Pull Request
7. Merged the feature branch into the `main` branch
